# Color Palette Generator

**#19 of Manzi's 100 GitHub Projects Roadmap**

A professional color palette generator with 5 palette algorithms, swatch locking, clipboard export, and palette history — all client-side, zero dependencies.

## Features

- **5 palette types:** Random, Monochromatic, Complementary, Analogous, Triadic
- Generates **5-color palettes** with large, clickable swatches
- Each swatch displays: HEX code, RGB values, HSL values
- **Click any swatch** to copy its HEX code (shows toast notification)
- **Lock individual swatches** (🔒) so they survive regeneration
- **Export button** — copies all 5 HEX codes as a comma-separated list
- **Palette history** — last 5 generated palettes shown as mini swatches
- **Spacebar** shortcut to generate a new palette
- Smooth hover transitions on all swatches

## Usage

Open `index.html` in your browser — no build step needed.
