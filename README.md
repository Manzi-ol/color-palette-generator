# Color Palette Generator

> Generate beautiful colour palettes with **5 harmony algorithms** — lock swatches, copy HEX codes, browse history.

## Features

- **5 colour harmony algorithms** — Random, Complementary, Analogous, Triadic, Monochromatic
- Lock individual swatches to keep them while regenerating the rest
- Click any swatch to copy its HEX code to clipboard
- Palette history — browse previously generated palettes
- Keyboard shortcut — press `Space` to generate a new palette
- Displays HEX, RGB, and HSL values for each colour
- Pure frontend — no backend, no dependencies
